üìã Usage Guidelines

When to Use Each Type

  • Dropdown Menu: Action lists (Edit, Delete, Share), user account menus, settings and options
  • Context Menu: Right-click actions, table row actions, file/folder operations
  • Select Menu: Form inputs requiring selection, filters and sorting, settings with predefined options
  • Combobox: Location/address search, tag selection, search with suggestions
  • Command Menu: Power user shortcuts, quick navigation, search across app
  • Mega Menu: Complex navigation hierarchies, e-commerce categories, feature discovery

Best Practices

  • Keep menu item labels concise and action-oriented
  • Group related items together
  • Use clear, descriptive section headers
  • Show keyboard shortcuts for common actions
  • Limit menu depth (avoid deep nesting)
  • Maintain consistent spacing and sizing
  • Use icons to aid recognition
  • Separate dangerous actions with dividers

‚ôø Accessibility

Keyboard Navigation

  • Tab: Focus trigger
  • Enter/Space: Open menu
  • Arrow Up/Down: Navigate items
  • Arrow Left/Right: Navigate submenus
  • Escape: Close menu
  • Home/End: First/last item
